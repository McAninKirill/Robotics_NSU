1. Команда bg предназначена для возобновления исполнения остановленной задачи в фоновом режиме
в командных оболочках bash и ksh.
2. Команда fg предназначена для перевода фоновой задачи в обычный режим в командных оболочках bash и ksh.
3. Команда ping в Linux посылает пакеты, пока вы не прервёте их отправку вручную с помощью сочетания клавиш Ctrl+C.
4. Команда scp используется для безопасного копирования файлов в удаленную систему и из нее по протоколу SSH.
5. Команда uptime выводит в одну строку информацию о работе системы. Она возвращает данные
о том, как долго работала Ваша система, о количестве пользователей, выполняющих сеансы, и средних значениях загрузки.
6. Команда uname выводит базовую системную информацию. Обычно она вызывается с
опцией -a для отображения всей доступной информации.
7. Команда pgrep ищет процессы на основе указанных критериев, таких как имя процесса,
идентификатор пользователя, идентификатор терминала и.т.д. 
8. Команда cal печатает календарь ASCII указанного месяца или года.
9. Команда bc действует как научный калькулятор с выполнением математических вычислений произвольной точности.
10. Команда tr заменяет каждый символ, содержащийся в String1 из стандартного ввода,
на символ в той же позиции в String2.
